VTUHub — VTU Scraper

This service fetches full VTU result pages and returns the complete HTML. Use the interactive API docs to call endpoints and view examples.

What this service returns

  • Full HTML for the VTU result page.
  • Automatic retry on invalid captcha (up to 5 attempts).
  • Two main endpoints: /single-post and /range-post.

If you want parsed JSON rather than raw HTML, add a post-processing step or request a parser endpoint.

Quick examples

Single USN — returns full result HTML for one USN. 

curl -X POST "http://127.0.0.1:8000/single-post" \
  -H "Content-Type: application/json" \
  -d '{"index_url":"https://results.vtu.ac.in/JJEcbcs25/index.php","usn":"4MH"}'

Range — returns a map of USN → HTML for a range.

curl -X POST "http://127.0.0.1:8000/range-post" \
  -H "Content-Type: application/json" \
  -d '{"index_url":"https://results.vtu.ac.in/JJEcbcs25/index.php","start_usn":"4MH","end_usn":"4MH"}'

Status & troubleshooting

  • Server: http://127.0.0.1:8000
  • If invalid captcha repeats, improve OCR or input captcha manually.
  • Ensure VTUScraper is imported correctly and server is run from project root.
